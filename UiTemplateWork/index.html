การติดตั้งและใช้งาน Daisy UI

Daisy UI เป็นปลั๊กอินสำหรับ Tailwind CSS ที่ช่วยให้การสร้าง UI สวยงามและรวดเร็วขึ้น คุณสามารถติดตั้งและใช้งานได้หลายวิธี:

1. ใช้ CDN (วิธีที่ง่ายที่สุด)

สำหรับโปรเจกต์ที่ไม่ต้องการการตั้งค่าที่ซับซ้อน คุณสามารถลิงก์ไฟล์ CSS ของ Daisy UI ได้โดยตรงในไฟล์ HTML ของคุณ:

<link href="https://cdn.jsdelivr.net/npm/daisyui@latest/dist/full.css" rel="stylesheet" type="text/css" />
<script src="https://cdn.tailwindcss.com"></script>

หมายเหตุ: วิธีนี้เหมาะสำหรับการทดลองหรือโปรเจกต์ขนาดเล็ก เนื่องจากไม่สามารถปรับแต่ง Tailwind หรือ Daisy UI ได้

2. ติดตั้งผ่าน npm (แนะนำ)

สำหรับโปรเจกต์ที่ต้องการความยืดหยุ่นและการปรับแต่งสูงสุด แนะนำให้ติดตั้งผ่าน Node.js และ npm:

  1. ติดตั้ง Tailwind CSS และ Daisy UI: 
    npm install -D tailwindcss daisyui
  2. เพิ่ม Daisy UI ในไฟล์ tailwind.config.js: 
    module.exports = {
  // ...
  plugins: [require("daisyui")],
}
  3. สร้างไฟล์ CSS หลัก:

    สร้างไฟล์ (เช่น app.css) และเพิ่ม directives ของ Tailwind:

    @tailwind base;
@tailwind components;
@tailwind utilities;
  4. คอมไพล์ CSS:

    ใช้ Tailwind CLI เพื่อคอมไพล์ CSS ของคุณ:

    npx tailwindcss -i ./app.css -o ./output.css --watch

    จากนั้นลิงก์ไฟล์ output.css ใน HTML ของคุณ
